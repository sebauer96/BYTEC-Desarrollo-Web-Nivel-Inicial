🔤 Tipografía: Familias de Fuentes

Las fuentes se agrupan en familias según su estilo. Cada una transmite una sensación distinta:

Aa Bb 123

Serif

Tienen "patitas" en las letras. Transmiten elegancia, tradición, formalidad.

Times, Georgia, Playfair Display
Aa Bb 123

Sans-Serif

Sin patitas, líneas limpias. Transmiten modernidad, simpleza, claridad.

Arial, Inter, Roboto
Aa Bb 123

Monospace

Todas las letras ocupan el mismo ancho. Para código, datos técnicos.

Courier, Space Mono
Aa Bb 123

Display

Decorativas, llamativas. Para títulos grandes y branding.

Bebas Neue, Lobster

🌐 Google Fonts: Tipografías Gratis

Google Fonts tiene más de 1500 tipografías gratuitas listas para usar. Solo necesitás 2 pasos:

1

Linkear en el HTML

Agregá el link en el <head>:

<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;700&display=swap" rel="stylesheet">
→
2

Usar en el CSS

Aplicá la fuente con font-family:

body {
  font-family: 'Inter', sans-serif;
}

h1 {
  font-family: 'Inter', sans-serif;
  font-weight: 700;
}

🌟 Andá a fonts.google.com, elegí una fuente, hacé click en "Get font" → "Get embed code" y copiá el link.

📝 Propiedades CSS de Texto

Más allá de la fuente, hay propiedades clave para que el texto se vea bien:

Propiedad ¿Qué hace? Ejemplo
font-size Tamaño del texto font-size: 18px
font-weight Grosor (100-900, normal, bold) font-weight: 700
line-height Altura entre líneas (1.5 = 150%) line-height: 1.6
letter-spacing Espacio entre letras letter-spacing: 2px
text-align Alineación (left, center, right) text-align: center
text-transform uppercase, lowercase, capitalize text-transform: uppercase

💡 Tip de legibilidad: usá line-height: 1.5 a 1.7 para párrafos. Es lo que hace que el texto sea cómodo de leer.

🪜 Jerarquía Visual

La jerarquía guía la mirada del usuario. Lo más importante debe verse más, lo secundario menos:

H1 — 48px / 900

Título principal

H2 — 32px / 700

Subtítulo de sección

H3 — 22px / 600

Subtítulo menor

P — 16px / 400

Párrafo de cuerpo. Este es el texto que el usuario va a leer más.

SMALL — 13px / 400 / muted Texto secundario, fechas, notas al pie.

📐 Regla 1.5x: cada nivel debe ser ~1.5 veces más grande que el anterior. Así se nota la diferencia visual.

💻 Práctica 1: Aplicar Google Fonts

Vamos a tomar una página simple y mejorarla con tipografías profesionales:

<!-- En el HEAD -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:wght@700&family=Inter:wght@400;600&display=swap" rel="stylesheet">
/* style.css */
body {
  font-family: 'Inter', sans-serif;
  font-size: 16px;
  line-height: 1.6;
  color: #333;
}

h1, h2, h3 {
  font-family: 'Playfair Display', serif;
  line-height: 1.2;
}

h1 { font-size: 48px; font-weight: 700; }
h2 { font-size: 32px; font-weight: 700; }
h3 { font-size: 22px; font-weight: 600; }

.uppercase {
  text-transform: uppercase;
  letter-spacing: 2px;
  font-size: 13px;
}

🎨 Colores en CSS

Hay varias formas de declarar colores en CSS. Las más usadas son:

Hexadecimal

#f75001

Lo más común en diseño. Empieza con # seguido de 6 caracteres (RGB en hex).

RGB

rgb(247, 80, 1)

Red, Green, Blue. Cada valor de 0 a 255.

RGBA

rgba(247, 80, 1, 0.5)

RGB + alpha (transparencia). El último valor va de 0 (transparente) a 1 (opaco).

HSL

hsl(20, 99%, 49%)

Hue (matiz), Saturation, Lightness. Más intuitivo para diseñar.

💡 Tip: usá rgba() cuando necesites transparencia (sombras, overlays). Para todo lo demás, hex es lo estándar.

🎨 Crear Paletas Armónicas

Una buena paleta tiene 5-7 colores que se complementan. La estructura típica:

Primario #f75001
Secundario #04589d
Acento #00e676
Fondo #060614
Superficie #131340
Texto #e8e8f0
Texto sutil #8888aa

Estructura de paleta

  • 1 color primario (marca)
  • 1 secundario (acompaña)
  • 1 acento (destacar acciones)
  • 2-3 grises (fondo, superficie, texto)

Herramientas para paletas

  • coolors.co — Genera paletas
  • realtimecolors.com — Preview en vivo
  • colorhunt.co — Paletas curadas
  • paletton.com — Color theory

⚙️ Variables CSS (Custom Properties)

Las variables CSS te permiten guardar valores y reusarlos. Perfectas para paletas de colores:

/* Definir las variables en :root (al inicio del CSS) */
:root {
  --color-primario: #f75001;
  --color-secundario: #04589d;
  --color-fondo: #060614;
  --color-texto: #e8e8f0;
  --color-muted: #8888aa;

  --fuente-titulo: 'Playfair Display', serif;
  --fuente-cuerpo: 'Inter', sans-serif;
}

/* Usarlas con var() */
body {
  background: var(--color-fondo);
  color: var(--color-texto);
  font-family: var(--fuente-cuerpo);
}

h1 {
  color: var(--color-primario);
  font-family: var(--fuente-titulo);
}

.btn {
  background: var(--color-primario);
}

⭐ ¿Por qué usar variables? Si querés cambiar el color primario de toda la web, lo cambiás en un solo lugar. Esencial para mantener código limpio.

👁️ Contraste y Accesibilidad

El contraste entre texto y fondo es crítico para que se pueda leer. WCAG (estándar de accesibilidad) recomienda:

Buen contraste

Texto legible en cualquier dispositivo

✅ Ratio: 13.4 (AAA)
Contraste medio

Solo para textos secundarios y notas

⚠️ Ratio: 4.6 (AA)
Mal contraste

No se lee, evitalo siempre

❌ Ratio: 1.5 (Fail)

🔍 Probalo: usá webaim.org/resources/contrastchecker para verificar el contraste de tus combinaciones.

🖼️ Imágenes en HTML & CSS

Hay 2 formas de insertar imágenes: como etiqueta HTML o como fondo CSS. Cada una tiene su uso:

<img> (HTML)

<img src="foto.jpg" alt="Persona sonriendo">

Usá esto cuando la imagen es contenido: fotos, logos, productos.

vs

background-image (CSS)

.hero {
  background-image: url('hero.jpg');
}

Usá esto cuando la imagen es decorativa: hero, patterns, banners.

📸 Imágenes gratis: unsplash.com, pexels.com y pixabay.com. Todas con licencia libre.

🌅 Propiedades de Background

El background tiene varias propiedades para controlar imágenes y colores de fondo:

.hero {
  background-color: #060614;                          /* Color de fondo */
  background-image: url('imagen.jpg');                /* Imagen */
  background-size: cover;                             /* Ajusta al contenedor */
  background-position: center;                        /* Centra la imagen */
  background-repeat: no-repeat;                       /* No repetir */
  background-attachment: fixed;                       /* Efecto parallax */
}

/* Atajo (shorthand) */
.hero {
  background: #060614 url('imagen.jpg') center/cover no-repeat;
}

/* Gradientes (degradados) */
.banner-gradient {
  background: linear-gradient(135deg, #f75001, #04589d);
}

/* Overlay oscuro sobre imagen */
.hero-overlay {
  background:
    linear-gradient(rgba(0,0,0,0.6), rgba(0,0,0,0.6)),
    url('imagen.jpg') center/cover;
}

🎨 El truco del overlay: apilar un gradiente oscuro sobre la imagen permite que el texto sea legible aunque la foto sea muy clara.

🌈 Gradientes (Degradados)

Los gradientes son transiciones suaves entre colores. Le dan profundidad y modernidad a tu diseño:

Linear horizontal

linear-gradient(to right, ...)

Linear diagonal

linear-gradient(135deg, ...)

Radial

radial-gradient(circle, ...)

Pastel suave

linear-gradient(135deg, ...)

🌟 Recurso: uigradients.com tiene 100+ gradientes listos para copiar.
