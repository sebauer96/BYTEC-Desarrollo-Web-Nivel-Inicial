😤 El Problema: Layouts sin Flexbox

Hasta ahora, para poner elementos al lado usamos display: inline-block. Pero tiene problemas:

Problemas de inline-block

  • Espacios fantasma entre elementos
  • Difícil centrar verticalmente
  • No se reparten el espacio sobrante
  • El orden depende del HTML

¿Qué queremos lograr?

  • Poner elementos lado a lado fácilmente
  • Centrar en ambos ejes (horizontal y vertical)
  • Distribuir el espacio equitativamente
  • Adaptar el layout a distintas pantallas

🎉 Flexbox resuelve todos estos problemas con unas pocas propiedades. ¡Vamos a verlo!

📐 ¿Qué es Flexbox?

Flexbox (Flexible Box Layout) es un sistema de CSS para distribuir elementos en una dimensión: en fila o en columna.

Contenedor (padre)
Hijo 1
Hijo 2
Hijo 3

Contenedor (padre)

El elemento al que le ponés display: flex. Controla cómo se distribuyen sus hijos.

Items (hijos)

Los elementos directos dentro del contenedor flex. Se alinean y distribuyen automáticamente.

⭐ Regla clave: Flexbox funciona con dos niveles: propiedades del padre (contenedor) y propiedades de los hijos (items).

🚀 display: flex — El Primer Paso

Con solo escribir display: flex en el padre, los hijos se ponen uno al lado del otro automáticamente:

.contenedor {
  display: flex;    /* ¡Eso es todo! Los hijos se ponen en fila */
}

↔️ Los Dos Ejes de Flexbox

Flexbox trabaja con dos ejes. Entenderlos es clave para dominar la alineación:

EJE PRINCIPAL (main axis) →
1
2
3
EJE CRUZADO ↓
(cross axis)

Eje Principal (Main Axis)

  • Dirección en la que van los hijos
  • Por defecto: horizontal (→)
  • Se controla con flex-direction
  • Se alinea con justify-content

Eje Cruzado (Cross Axis)

  • Perpendicular al eje principal
  • Por defecto: vertical (↓)
  • Se alinea con align-items
  • Siempre es el "otro" eje

🧭 flex-direction

Define la dirección del eje principal (cómo se acomodan los hijos):

.contenedor {
  display: flex;
  flex-direction: row;            /* Fila → (por defecto) */
  flex-direction: row-reverse;    /* Fila invertida ← */
  flex-direction: column;         /* Columna ↓ */
  flex-direction: column-reverse; /* Columna invertida ↑ */
}

↔️ justify-content

Controla la distribución de los hijos a lo largo del eje principal:

.contenedor {
  display: flex;
  justify-content: flex-start;      /* Al inicio (defecto) */
  justify-content: center;          /* Centrado */
  justify-content: flex-end;        /* Al final */
  justify-content: space-between;   /* Espacio entre elementos */
  justify-content: space-around;    /* Espacio alrededor */
  justify-content: space-evenly;    /* Espacio parejo */
}

flex-end

space-around

↕️ align-items

Controla la alineación de los hijos en el eje cruzado (perpendicular al principal):

.contenedor {
  display: flex;
  align-items: stretch;      /* Se estiran para llenar (defecto) */
  align-items: flex-start;   /* Arriba */
  align-items: center;       /* Centro */
  align-items: flex-end;     /* Abajo */
  align-items: baseline;     /* Alineados por la línea de texto */
}
stretch
flex-start
center
flex-end

🧠 Truco mnemotécnico: justify = eje principal (la J parece una flecha →). align = eje cruzado (perpendicular).

✨ El Centrado Perfecto

El centrado perfecto (horizontal y vertical) fue durante años un dolor de cabeza. Con Flexbox se logra con 3 líneas:

.contenedor {
  display: flex;
  justify-content: center;    /* Centra en el eje principal → */
  align-items: center;        /* Centra en el eje cruzado ↓ */
  height: 100vh;              /* Ocupa toda la pantalla */
}
¡Centrado!

🎯 Este patrón es super común: lo vas a usar para centrar modales, mensajes de error, loaders, héroes de landing pages, etc.

📏 gap — Espacio entre Hijos

La propiedad gap agrega espacio entre los hijos, sin agregar espacio extra al inicio ni al final:

.contenedor {
  display: flex;
  gap: 20px;              /* Espacio uniforme entre todos los hijos */
}

/* También podés separar fila y columna */
.contenedor {
  display: flex;
  flex-wrap: wrap;
  gap: 20px 30px;         /* 20px entre filas, 30px entre columnas */
}

Con margin

Tenés que sacar el margin del último hijo. Complejo.

margin ← 😤 → margin
vs

Con gap

Solo agrega espacio entre elementos. Simple y limpio.

gap ← 😎 → gap

💡 Tip: Usá gap en vez de margin entre hijos de un contenedor flex. Es más limpio y predecible.

🔄 flex-wrap — Saltar de Línea

Por defecto, los hijos se comprimen para entrar en una sola línea. Con flex-wrap les permitís saltar a la siguiente línea:

.contenedor {
  display: flex;
  flex-wrap: nowrap;    /* No salta de línea (defecto) — se comprimen */
  flex-wrap: wrap;      /* Salta a la siguiente línea si no entra */
  flex-wrap: wrap-reverse;  /* Salta pero en dirección inversa */
}

👶 Propiedades de los Hijos

Hasta ahora vimos propiedades del padre. Ahora vamos a ver las propiedades que se aplican a cada hijo individual:

Propiedad ¿Qué hace? Ejemplo
flex-grow Cuánto crece para llenar espacio sobrante flex-grow: 1
flex-shrink Cuánto se achica si falta espacio flex-shrink: 0
flex-basis Tamaño base antes de crecer/achicarse flex-basis: 200px
align-self Alineación individual (override del padre) align-self: flex-end
order Cambia el orden visual sin tocar el HTML order: -1

📈 flex-grow — Crecer para Llenar

flex-grow define cuánto crece un hijo para ocupar el espacio sobrante del contenedor:

/* Todos los hijos crecen igual */
.hijo {
  flex-grow: 1;     /* Se reparten el espacio sobrante por igual */
}

/* El hijo-2 crece el doble que los demás */
.hijo-1 { flex-grow: 1; }
.hijo-2 { flex-grow: 2; }   /* Ocupa el doble de espacio extra */
.hijo-3 { flex-grow: 1; }

1
1
1
Hijo del medio: flex-grow: 2
1
2
1

💡 flex-grow: 0 (defecto) = no crece. flex-grow: 1 = ocupa todo el espacio sobrante disponible.

⚡ El Shorthand: flex

La propiedad flex es un atajo que combina flex-grow, flex-shrink y flex-basis en una sola línea:

/* flex: grow shrink basis */
.hijo {
  flex: 0 1 auto;     /* Valores por defecto */
}

/* Atajos comunes */
.hijo {
  flex: 1;            /* = flex: 1 1 0%  → Crece y se achica por igual */
}

.hijo {
  flex: none;         /* = flex: 0 0 auto → Tamaño fijo, no cambia */
}

.sidebar  { flex: 0 0 250px; }   /* Sidebar fijo de 250px */
.main     { flex: 1; }            /* Contenido principal flexible */

Valores más usados

  • flex: 1 — Crece para llenar
  • flex: none — Tamaño fijo
  • flex: 0 0 200px — Fijo 200px
  • flex: 2 — Crece el doble

💻 Práctica 2: Navbar con Flexbox

Vamos a construir una barra de navegación profesional usando Flexbox:

<nav class="navbar">
  <div class="logo">MiSitio</div>
  <ul class="nav-links">
    <li><a href="#">Inicio</a></li>
    <li><a href="#">Servicios</a></li>
    <li><a href="#">Contacto</a></li>
  </ul>
  <button class="btn-cta">Registrarse</button>
</nav>
/* El truco: flex en el nav, y flex: 1 en los links */
.navbar {
  display: flex;
  align-items: center;
  padding: 0 24px;
  height: 60px;
  background-color: #0d0d2b;
  border-bottom: 2px solid #1a1a5e;
}

.nav-links {
  display: flex;
  list-style: none;
  gap: 24px;
  flex: 1;               /* Ocupa todo el espacio del medio */
  justify-content: center;
}
