📋 Formularios: La Puerta de Entrada

Un formulario es un bloque de HTML que permite al usuario ingresar datos. Todo formulario empieza con la etiqueta <form>.

<form>
  <!-- Aquí van todos los campos del formulario -->
</form>

¿Por qué usar <form>?

  • Agrupa todos los campos relacionados
  • Permite enviar datos al servidor
  • Mejora la accesibilidad del sitio
  • El navegador sabe que es un formulario

Regla de oro 🏆

Todos los campos de entrada deben estar dentro de un <form>. Si quedan afuera, no se enviarán.

🏷️ La dupla: <label> + <input>

Cada campo necesita dos cosas: un nombre visible (label) y una caja para escribir (input).

<label for="nombre">Tu nombre:</label>
<input type="text" id="nombre" placeholder="Ej: María">

Anatomía de la conexión:

for="nombre" en el <label>
══ se conecta con ══
id="nombre" en el <input>

💡 ¿Por qué importa? Al hacer clic en el label, el cursor salta al input. Esto mejora la accesibilidad (lectores de pantalla) y la usabilidad (más fácil de usar en celulares).

⌨️ Tipos de Input

El atributo type cambia el comportamiento del campo. ¡Probá cada uno en tu celular!

type="text"

Texto libre. Para nombres, direcciones, etc.

type="email"

Valida formato de email. El teclado móvil muestra @.

type="password"

Oculta los caracteres con puntos (•••).

type="number"

Solo acepta números. Teclado numérico en móvil.

📱 Dato: En celulares, type="email" muestra un teclado con @, y type="number" muestra el teclado numérico. ¡El navegador es inteligente!

📦 Select, Option y Button

Lista desplegable

<label for="pais">País:</label>
<select id="pais">
  <option value="">Elegí...</option>
  <option value="ar">Argentina</option>
  <option value="mx">México</option>
  <option value="co">Colombia</option>
</select>

Usá <select> cuando hay varias opciones predefinidas. Cada <option> tiene un value (dato real) y un texto visible.

El botón de envío

<button type="submit">
  Registrarme
</button>

type="submit" dispara el envío del formulario. Es el "botón de acción" que el usuario presiona al terminar.

⚡ Sin un botón de tipo submit, el formulario no puede enviarse con un clic.

🔧 Atributos útiles de los inputs

Estos atributos agregan comportamiento extra a tus campos:

Atributo ¿Qué hace? Ejemplo
placeholder Texto de ayuda dentro del campo placeholder="Ej: Juan"
required Campo obligatorio, no se puede enviar vacío required
name Nombre del dato al enviarse name="email"
value Valor por defecto del campo value="Argentina"
maxlength Máximo de caracteres permitidos maxlength="50"
min / max Rango para inputs numéricos min="1" max="100" 
<input type="email" id="correo" name="correo"
       placeholder="tu@email.com" required maxlength="100">

💻 Práctica Guiada 1: Armá tu Formulario

Vamos a codear juntos. Creá un formulario de registro con estos campos:

✅ Nombre completo type="text"
✅ Correo electrónico type="email"
✅ Contraseña type="password"
✅ País <select> con al menos 3 opciones
✅ Botón "Registrarme" type="submit"
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Registro BYTEC</title>
</head>
<body>
  <h1>Crear Cuenta</h1>
  <form>
    <label for="nombre">Nombre completo:</label>
    <input type="text" id="nombre" placeholder="Ej: María García" required>

    <label for="correo">Correo electrónico:</label>
    <input type="email" id="correo" placeholder="tu@email.com" required>

    <label for="pass">Contraseña:</label>
    <input type="password" id="pass" placeholder="Mínimo 6 caracteres" required>

    <label for="pais">País:</label>
    <select id="pais" required>
      <option value="">Seleccioná tu país</option>
      <option value="ar">Argentina</option>
      <option value="mx">México</option>
      <option value="co">Colombia</option>
      <option value="cl">Chile</option>
      <option value="pe">Perú</option>
    </select>

    <button type="submit">Registrarme</button>
  </form>
</body>
</html>

🎨 CSS: Separando Estructura de Diseño

🦴

HTML = Esqueleto

Define qué hay en la página: títulos, párrafos, formularios, imágenes.

+
👕

CSS = Ropa

Define cómo se ve: colores, tamaños, tipografías, espaciados.

=
🌐

Página Web

Estructura + diseño = una web completa y profesional.

📏 Buena práctica: Nunca mezcles estilos dentro del HTML (evitar style="..."). Siempre usá un archivo .css separado.

🔗 Conectar CSS con HTML

Creamos un archivo style.css y lo vinculamos en el <head> del HTML:

<!-- Dentro del <head> de tu HTML -->
<head>
  <meta charset="UTF-8">
  <title>Mi Página</title>
  <link rel="stylesheet" href="style.css">
</head>
1

Creá un archivo llamado style.css en la misma carpeta que tu HTML.

2

Agregá la etiqueta <link> dentro del <head>.

3

El atributo href apunta a tu archivo CSS.

4

¡Listo! Todo lo que escribas en el .css se aplicará al HTML.

📐 Sintaxis de CSS

Toda regla CSS sigue esta estructura universal:

SELECTOR h1
{
PROPIEDAD color : VALOR blue ;
PROPIEDAD font-size : VALOR 32px ;
}
/* Esto selecciona TODOS los h1 y les cambia el color */
h1 {
  color: blue;
  font-size: 32px;
}

/* Esto selecciona TODOS los párrafos */
p {
  color: gray;
  font-family: Arial, sans-serif;
}

🎯 Selectores CSS

Los selectores le dicen a CSS a qué elementos aplicar los estilos:

Por etiqueta

body {
  background-color: #1a1a2e;
}

h1 {
  color: orange;
}

Aplica a todas las etiquetas de ese tipo.

Por clase ⭐ MÁS USADO

/* En CSS: se escribe con punto */
.boton-principal {
  background-color: #f75001;
  color: white;
}
<!-- En HTML: se usa el atributo class -->
<button class="boton-principal">
  Enviar
</button>

Reutilizable. Podés aplicar la misma clase a muchos elementos.

💡 Convención: Los nombres de clase se escriben en kebab-case (todo minúsculas, separado por guiones): boton-principal, texto-grande, card-info.

🎨 Propiedades CSS Comunes

Colores

color: #f75001;
background-color: #04589d;

Color de texto y de fondo. Usá códigos hexadecimales (#) o nombres como red, white.

Tipografía

font-family: Arial, sans-serif;
font-size: 18px;
font-weight: bold;

Familia tipográfica, tamaño y peso (grosor) del texto.

Espaciado

padding: 10px;
margin: 20px;

Padding: espacio interno. Margin: espacio externo.

Bordes y Redondeo

border: 2px solid #f75001;
border-radius: 8px;

Bordes visibles y esquinas redondeadas.

💻 Práctica Guiada 2: Pintando la Web

Vamos a darle vida al formulario de la Práctica 1 con CSS:

🎨 Color de fondo al body → background-color
✏️ Cambiar tipografía → font-family
🔤 Color de texto a los h1 y label → color
🔘 Crear clase .boton-principal para el botón
📦 Agregar padding y margin para espaciar
/* style.css */
body {
  background-color: #0a0a23;
  font-family: Arial, sans-serif;
  color: white;
  padding: 40px;
}

h1 {
  color: #f75001;
}

label {
  display: block;
  margin-top: 15px;
  font-weight: bold;
}

input, select {
  width: 100%;
  padding: 10px;
  margin-top: 5px;
  border: 2px solid #04589d;
  border-radius: 5px;
  font-size: 16px;
}

.boton-principal {
  background-color: #f75001;
  color: white;
  padding: 12px 30px;
  border: none;
  border-radius: 5px;
  font-size: 18px;
  margin-top: 20px;
  cursor: pointer;
}

.boton-principal:hover {
  background-color: #d44400;
}
